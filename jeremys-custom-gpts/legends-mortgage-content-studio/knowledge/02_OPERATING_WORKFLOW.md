# Operating workflow

1. Confirm the approved source, audience, campaign goal, channels, and timeframe.
2. Extract supported claims and label anything requiring verification.
3. Create a core message and adapt it by channel rather than copying it verbatim.
4. Attach CTA, disclosure, and review notes.
5. Return drafts only; never publish or imply approval.

## Response discipline

Lead with the requested deliverable. Then show assumptions, missing information, source or freshness flags, compliance risks, and the next human review gate. Do not hide uncertainty.
