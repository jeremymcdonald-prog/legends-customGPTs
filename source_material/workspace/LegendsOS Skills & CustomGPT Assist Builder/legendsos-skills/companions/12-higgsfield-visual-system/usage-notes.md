# Usage Notes — Higgsfield Visual System

## When to use Higgsfield

- Cinematic mortgage visuals (closing day, broker handoff, market metaphor).
- Florida lifestyle scenes (Northeast Florida coast, suburb, downtown).
- Premium creator assets (Reel cover plates, YT thumbnail face plates).
- Motion backgrounds for long form intros and outros.
- Pattern-interrupt visuals for short-form hooks.
- Image-to-video hooks (still that animates into the first 2 seconds of a Reel).

## When to use Canva instead

- Anything that is text-first (carousels, motivational quotes, education slides).
- GBP posts (Canva handles the local angle template).
- Anything that needs a precise compliance footer (Canva holds the footer locked).
- Anything that needs to ship in 60 seconds (Higgsfield's render time costs).

## What this skill needs from you

- Use case (still, image-to-video, B-roll plate, motion background).
- Concept (one sentence: what you want to feel).
- Aspect (1080x1350, 1080x1920, 1280x720, 16:9, etc.).
- Reserved text zone (where Canva text will sit).
- Brand engine (Cinematic Authority, Pattern Interrupt, Educational Minimal).

## What you get back

A four-layer prompt block, the default negative prompt, the style preset, request_id placeholders, the asset filename, and a Concept Gate status.

## What the skill never does

- Generate without Concept Gate clearance (for live runs).
- Render text inside the visual.
- Mark assets approved.
- Push to a public channel.
