# Agent 13 — Realtor Newsletter Agent

## What This Is

A reusable agent module for generating Realtor-facing newsletters, social content, and AI-assisted marketing on behalf of Jeremy McDonald and The Legends Mortgage Team powered by Loan Factory.

This folder contains everything needed to deploy this agent as a skill, custom GPT instruction file, or standalone module inside the LegendsOS system.
