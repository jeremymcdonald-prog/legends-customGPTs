# LegendsOS Story Bank

Real source material in. Anonymized lessons out. Source IDs other skills pull from.

## Files

- `SKILL.md`, `CUSTOM_GPT.md`, `compliance.md`, `usage-notes.md`, `test-prompts.md`
- `knowledge/` — schema, review rules, privacy, frameworks, mapping, risk scoring
- `examples/` — sample source records and lesson summaries

## Source ID prefixes

- `ZILLOW_REVIEW_SOURCE_xxx`
- `GBP_REVIEW_SOURCE_xxx`
- `BORROWER_MEMORY_xxx`
- `REALTOR_SCENARIO_xxx`
- `WEBSITE_BLOG_SOURCE_xxx`

## Hard rules

- No invented customers.
- No invented reviews.
- No names without written permission.
- No addresses without deal-public + approval.
- No rate / payment / credit / income numbers.
- Lesson framing over testimonial framing.

## Sits inside

Story Bank is the trust layer for every other content skill. If a story post lacks a source ID from this bank, it does not ship.
