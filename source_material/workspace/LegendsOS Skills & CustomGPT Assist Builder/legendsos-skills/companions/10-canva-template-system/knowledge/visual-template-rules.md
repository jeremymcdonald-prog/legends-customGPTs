# Visual Template Rules

References `_shared/brand-dna/visual-dna.md`. Template-specific rules below.

## Palette enforcement

- Charcoal `#1A1A1A` and Black `#0A0A0A` as base.
- Loan Factory Orange `#F26A21` OR Champagne Gold `#C9A24B` as accent. Not both fighting.
- White `#FFFFFF` headlines. Silver `#C7C7C7` support.

## Typography

- Headline: Anton or Archivo Black.
- Sub: Montserrat SemiBold.
- Premium accent: elegant script for brand moments only.
- No third type family ever.

## Safe zones

| Canvas | Safe zone (no critical text or face) |
|---|---|
| 1080x1350 (IG Feed) | Top 8%, bottom 12% |
| 1080x1920 (IG Reel cover, Story, YT Short) | Top 12% (avatar / handle overlay), bottom 18% (caption + UI overlay) |
| 1200x900 (GBP) | Bottom 14% reserved for body line + NMLS strip |
| 1200x1200 (LinkedIn) | Bottom 12% reserved for NMLS strip |
| 1280x720 (YT Long thumbnail) | Bottom 10% reserved for NMLS micro line |

## Footer placement rules

- Always last layer.
- Always readable on mobile.
- Silver text on dark backgrounds.
- Minimum font size: 18 px at 1080-wide canvas.

## One key word in accent

Per Visual DNA — one accent word per image. Examples:
- "WE closed in a WEEK" → "WE" accent.
- "BROKERS shop, BANKS sell" → "BROKERS" accent.
- "ZERO down. NO PMI." → "ZERO" accent.

## Image plate placement

When the template has an image plate (e.g., from Higgsfield), the plate occupies a defined region with a reserved text zone above or beside it. The plate must not bleed into the footer strip.

## Forbidden visual moves

- Stock photo broker imagery (handshake with cartoon house).
- Chrome word-art logos.
- Rainbow gradient backgrounds.
- Clip-art icons.
- Cartoon mascots.
- Bank logos.
- AI-looking faces (uncanny valley).
- Heavy filters that obscure legibility.
