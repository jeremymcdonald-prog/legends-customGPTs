## Versioning rule

`_v1` is the current production version. New versions append `_v2`, `_v3`. Old versions stay readable for asset auditing.

## When to add a new template

When a new pillar emerges or the visual DNA evolves. Each addition gets a spec in the next section's field schema doc.
