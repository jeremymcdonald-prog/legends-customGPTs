# Video Agent Prompt Rules

How the skill assembles the prompt block.

## Order of assembly

1. Identify lane (education / guidance / market / story / personal).
2. Pick persona that matches lane.
3. Pick length (short, mid, long).
4. Write hook (first 1.5 to 2 seconds — earns the watch).
5. Write setup (one sentence).
6. Write body (the teach or the story).
7. Write land (the takeaway).
8. Write CTA (one).
9. Generate on-screen text (3 to 5 lines).
10. Pick B-roll cues.
11. Pick thumbnail template + headline.
12. Run compliance gate.

## Hook tests

- Does it stop the scroll in 1.5 seconds?
- Does it state the payoff before it teaches?
- Does it avoid clickbait that won't be delivered?
- Does it match the persona's energy?

## Pacing checks

- Sentences ≤ 14 words on average.
- No em dashes.
- No filler ("just," "really," "basically").
- One thought per cut.

## Persona pacing reminders

- Inside Man: medium, calm authority. Pause on key terms.
- Coach: brisk, punchy. Directive.
- Wolf: sharp opening, slow takeaway. Confident.

## When to swap persona mid-batch

- Education week leans Inside Man (Mon, Wed, Fri).
- Guidance week leans Coach (Tue, Thu).
- Market week leans Wolf (Wed market update).
- Story or personal: persona picked by lane fit.

## Editing checks

- Watch the first 3 seconds back. If you'd swipe, rewrite.
- Read the script out loud. If your breath runs out, break the sentence.
- Confirm no compliance flag fires.
