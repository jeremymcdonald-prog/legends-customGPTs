# Weekly Video Batch Rules

## Weekly target

- 5 to 7 short form videos (Reel / TikTok / YT Short shareable cuts).
- 2 long form videos (YT Long), Tuesday and Thursday.
- 1 weekly recap (short or mid-form).
- 0 to 1 personal brand short.

## Batch production flow

1. Engine locks the week's video topics and pillar tags.
2. HeyGen skill emits prompt blocks for each video in order: long form first, shorts after, recap last.
3. Operator renders in HeyGen, downloads.
4. Thumbnails get filled in Canva.
5. Files land in operator drive, then media URL tracker, then HeroPost.

## Persona rotation in a batch

- Day 1 (Mon): Coach (broker advantage)
- Day 2 (Tue): Inside Man (education long form + short)
- Day 3 (Wed): Wolf (market update) + Coach (story short)
- Day 4 (Thu): Inside Man (long form) + Coach (story short)
- Day 5 (Fri): Coach (closing day) + Jeremy on camera (personal brand)
- Day 6 (Sat): Inside Man (Realtor AI tip)
- Day 7 (Sun): Optional reflection / Jeremy on camera

## Resource budget per batch

- HeyGen render minutes — track per video, reserve buffer.
- Canva thumbnail time — estimate 5 minutes per thumbnail with templates.
- Operator review time — 10 minutes per video.
- Total weekly burn budgeted in the Weekly Production Engine.

## Quality gate

Before the week's videos move to Scheduled in the tracker:

- All compliance flags clear.
- All thumbnails generated.
- All hosted URLs in the media URL tracker.
- Captions written (or HeyGen-generated + reviewed).
- Descriptions written for YT Long and YT Short.

## Fallback

If HeyGen API or service is down:

- Jeremy on camera for must-ship videos.
- Skip non-essential bulk videos and reweight the week.
- Document the gap in the production readiness report.
