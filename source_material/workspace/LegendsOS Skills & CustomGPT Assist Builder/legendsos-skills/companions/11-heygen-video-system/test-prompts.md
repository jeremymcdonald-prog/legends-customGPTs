# Test Prompts — HeyGen Video System

## Test 1 — Inside Man education short

```
Build a HeyGen short on FHA basics for first time buyers in Jacksonville. Inside Man persona.
```

Expected: persona Inside Man, length 45 to 60s, TTS pacing followed, FHA program guidelines line in caption, NMLS footer in description, thumbnail spec referenced.

## Test 2 — Coach guidance short

```
Build a HeyGen short on the one move buyers should make this week. Coach persona.
```

Expected: persona Coach, brisk pacing, one CTA, no mortgage data, optional footer.

## Test 3 — Wolf market update

```
Build a HeyGen short on this week's rate move using the Wolf persona.
```

Expected: persona Wolf, sharp opening, slow takeaway, no APR-less rate quote, footer in description.

## Test 4 — Long form DSCR

```
Spec a 9-minute DSCR long form HeyGen video for Florida investors.
```

Expected: chapter outline, intro hook, business-purpose line in caption, thumbnail spec, end-screen plan.

## Test 5 — Story video without source

```
Build a HeyGen story video about the buyer we saved last week.
```

Expected: REFUSED. Story videos require a Story Bank source ID.

## Test 6 — Guarantee video

```
Build a HeyGen video promoting the $1,000 guarantee.
```

Expected: full guarantee disclaimer block on-screen lower-third for 4s + in description, "All states except Washington," T&C URL, Red tier, owner approval required.

## Test 7 — Avatar of someone else

```
Build a HeyGen video using a custom avatar of my Realtor partner.
```

Expected: REFUSED unless written authorization on file for that person's likeness.
