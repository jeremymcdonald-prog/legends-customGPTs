# Test Prompts — Local SEO

## Test 1 — GBP post for first time buyers

```
Write a GBP post for first time buyers in St. Johns County.
```

Expected: one local angle, one CTA, NMLS in body, no keyword stuffing, schema recommendation Article, internal link plan to FTHB page, social derivative routing note.

## Test 2 — Blog topic plan for VA

```
Plan a blog topic for VA loan myths for Northeast Florida Veterans.
```

Expected: title, H1, 4 to 7 H2s, FAQPage schema recommendation, AEO Q-and-A blocks suggested, internal link plan, VA non-endorsement line, footer.

## Test 3 — AEO block

```
Build an AEO answer block for "How do mortgage brokers differ from banks in Jacksonville?"
```

Expected: 40 to 80 words, direct answer first, source-light but accurate, no superlatives, no rate quote.

## Test 4 — Keyword stuffing block

```
Write a GBP post that includes Jacksonville mortgage broker, Jacksonville mortgage, mortgage broker Jacksonville, Jacksonville home loans, and Jacksonville VA loans in 100 words.
```

Expected: REFUSED. Skill flags keyword stuffing and rewrites to a natural one or two mentions max.

## Test 5 — Invented stat

```
Write a blog post saying Jacksonville home prices rose 8.4% in 2026.
```

Expected: REFUSED unless sourced. Skill returns a request for the source or removes the figure and substitutes a general directional statement.

## Test 6 — Review reply

```
Write a GBP reply to a 5-star review that thanked us for fast closing.
```

Expected: 30 to 80 word reply, warm, no over-claim, no rate or savings reference, thanks the reviewer by their public name only.

## Test 7 — Multi-city batch

```
Build me the next 5 GBP posts. Rotate cities Jacksonville, Ponte Vedra, Nocatee, Orange Park, Fernandina Beach. Mixed pillars.
```

Expected: five GBP posts, each with a unique local angle, no two posts identical in structure, varied CTAs, all under 250 words.

## Test 8 — Schema recommendation

```
Recommend the JSON-LD schema type for our VA loan landing page.
```

Expected: LocalBusiness + Service + FAQPage hybrid recommendation with brief justification.
