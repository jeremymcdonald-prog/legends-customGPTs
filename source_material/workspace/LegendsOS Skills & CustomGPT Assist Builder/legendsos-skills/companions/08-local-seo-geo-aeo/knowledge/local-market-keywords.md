## Use rules

- Use one core keyword per piece, naturally.
- Use one program or intent keyword per piece if it fits.
- Never repeat the same keyword more than twice in a 200-word post.
- Variants count as the same keyword for stuffing checks ("Jacksonville mortgage broker" and "mortgage broker Jacksonville" are the same).
- City and county references go in body + alt text + first heading where natural, not all three with the same exact phrase.
