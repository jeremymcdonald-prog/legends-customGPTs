# NMLS + Equal Housing Footer (canonical)

## Standard lockup order

```
Legends Mortgage Team | Powered by Loan Factory
```

## Standard compliance strip

```
Equal Housing Lender · NMLS #1195266 · Loan Factory NMLS #320841
```

## Field reference

| Field | Value | Notes |
|---|---|---|
| Loan Factory Company NMLS | `#320841` | Six digits. Always verify complete before publish. Common error: `#32841` (missing digit) is invalid. |
| Jeremy McDonald LO NMLS | `#1195266` | Required when content is attributed to Jeremy or features him |
| Equal Housing tag | `Equal Housing Lender` (or `Equal Housing Opportunity` depending on piece) | Must be available on any public marketing piece |
| Service area | Northeast Florida (primary) | Used in local GBP / GEO copy |
| Contact for creative attribution | (904) 442-3213 · mcdonald-mtg.com | When piece is signed by Jeremy |

## "Sent on behalf of" rule

External email under the Loan Factory name requires NMLS #320841 + EHO footer **and** "Sent on behalf of [internal owner]" when the technical sender is not the true sender (e.g. AI-drafted lender AE outreach).

## Hard rule

Footer is non-negotiable on every public-facing creative, email, post, or graphic. Templates ship with the footer locked. Removing it is not an option in any skill, template, or downstream tool.
