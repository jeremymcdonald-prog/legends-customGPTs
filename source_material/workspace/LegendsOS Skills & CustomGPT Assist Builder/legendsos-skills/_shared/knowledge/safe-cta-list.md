# Safe CTA Library

One CTA per piece. Orange chip or underline in visuals. Never stack CTAs. Every customer deliverable ends with a clear next step.

## Comment / DM CTAs

- Comment "GUIDE"
- Comment "CHECKLIST"
- Comment "CREDIT"
- Comment "FHA"
- Comment "DPA"
- DM "NUMBERS"
- DM "VA"
- DM "INVESTOR"
- DM "RATES"
- DM "REFI"

## Soft brand CTAs

- Save this
- Share with a buyer
- Share with a Realtor
- Want the buyer checklist?
- Want the Realtor version?
- Ask me for today's numbers
- Send me the address

## Action CTAs

- Book a strategy call
- Start here
- Register
- Apply now (when context is appropriate and disclaimers fit)

## Authority post CTAs

- Read the comments
- Follow for the breakdown

## What NOT to use

- "Click here" (vague)
- "Learn more" (lazy)
- Anything that implies guaranteed approval
- Anything that offers a thing of value in exchange for referrals (RESPA)
- Stacked CTAs (pick one)

## CTA-to-engine matching

| Engine | Default CTA flavor |
|---|---|
| Cinematic Authority | Soft brand (Save this / Follow) or no CTA |
| Mortgage Education | Comment keyword (GUIDE, CHECKLIST, CREDIT) |
| Realtor Attraction | "Want the Realtor version?" / "Book a strategy call" |
| AI Authority | "Read the comments" |
| Veteran/VA | DM "VA" |
| Investor | DM "INVESTOR" |
| Webinar Promotion | "Register" |
| Market Update | "Ask me for today's numbers" |
| Team Momentum | Optional / no CTA |
| Pattern Interrupt | "Start here" |
