# FLO Document Rules

## Naming Convention

DocumentType_FirstInitialLastName_Source_Date.ext

Examples:
- W2_JSmith_Amazon_2025.pdf
- Paystub_JSmith_Amazon_2026-05-15.pdf
- BankStatement_JSmith_Chase_2026-04.pdf

## Rules
1. Never overwrite originals
2. Always preserve email attachments
3. Store OCR text separately
4. Use confidence scores
5. Escalate low confidence docs
6. Detect duplicates
7. Maintain audit logs
8. Never delete raw files
