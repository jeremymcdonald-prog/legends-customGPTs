# Ralph Loop

Use this loop for every work session.

Ralph means:

- Read
- Analyze
- List
- Prioritize
- Hammer
- Handoff

## 1. Read

Read:

- README.md
- 00_AI_CEO_MODE.md
- 01_SOURCE_OF_TRUTH.md
- 02_ARCHITECTURE.md
- 03_MODULE_REQUIREMENTS.md
- 09_BUILD_SEQUENCE.md
- STATUS.md if it exists
- HANDOFF.md if it exists

## 2. Analyze

Inspect current repo state.

Answer:

- What exists?
- What is missing?
- What is broken?
- What is the next highest value vertical slice?

## 3. List

Create or update a todo list.

Group by:

- now
- next
- later
- blocked

## 4. Prioritize

Pick one vertical slice.

A vertical slice must include:

- data model
- API/server action
- UI
- usage log if applicable
- test

## 5. Hammer

Execute.

Do not drift.

Do not audit forever.

## 6. Handoff

Write:

- what changed
- what was verified
- what failed
- what is next
- exact commands or files changed
