# Shared Compliance Engine

This module owns reusable mortgage advertising, RESPA, Fair Housing, Equal Housing, licensing, NMLS, rate, guarantee, AI disclosure, privacy, consent, prohibited-statement, and escalation rules. Every package inherits `compliance_rules.yaml` and the single checklist in `compliance_checklist.md`.

Profile-specific disclosure text is generated by the Identity Pack. Compliance rules do not replace current Loan Factory review or legal advice.
