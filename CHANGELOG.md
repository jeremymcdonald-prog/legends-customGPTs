# Changelog

All notable changes to the Legends Custom GPT Factory are documented here. The project uses semantic versioning.

## [1.0.0] - 2026-07-17

### Added

- Centralized, platform-neutral GPT Factory architecture.
- Identity Pack and profile inheritance for Jeremy, loan officers, Realtors, and team-shared ownership.
- Nine-profile Audience Engine.
- Shared Compliance and Conversation engines.
- Central Routing Engine and eighteen-entry CTA library.
- Reusable, non-deployed lead-capture architecture with consent, privacy, security, mapping, routing, fallback, OpenAPI, and tests.
- Deterministic GPT package generation with dependency fingerprints.
- Validated portfolio-wide rebuild workflow.
- Factory, profile, routing, deduplication, and package-generation validation.
- Multi-platform export preparation for eight assistant targets.
- Deterministic, metadata-driven root README generation.
