# Platform export layer

The core build graph is platform-neutral. `platforms.yaml` describes the export adapter for OpenAI GPTs, Claude Projects, Gemini Gems, Microsoft Copilot Agents, Open WebUI Assistants, AnythingLLM, OpenHands, and local LLM assistants.

Future platform-specific formatting belongs here. Identity, audience, compliance, routing, conversation, CTA, lead capture, knowledge, and tests must remain unchanged.
